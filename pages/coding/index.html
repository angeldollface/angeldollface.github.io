MY PROGRAMMING PROJECTS


MY DART PROJECTS

Here are three projects I’ve developed in the Dart programming language.

HARPO

Harpo is a small library to numerically decrypt/encrypt letters, words, and phrases and also offer conversion methods between number systems of different bases. Link: GitHub

TERM STYLE

I always thought that the output of command-line apps was just a slew of text and a little bit boring. So I decided to spice things up a little! Term Style gives you the ability to use emojis and colors in your terminal! Link: GitHub

SECURITY CHECK

One single person who is fairly interconnected on several platforms that live online probably has dozens upon dozens of passwords. How does an individual make sure that their passwords are secure and strong? They can’t, not with 100% certainty. Security Check attempts to solve this problem for (app) developers, who want to implement such a feature in a project they are building. Link: GitHub

MY JAVASCRIPT PROJECTS

Here is a selection of three libraries that I’ve developed in Javascript.

ETH PAY

ETH Pay is a simple library for people who would like to receive Ethereum tokens into their wallets and embed this functionality into their websites. The project is built in such a way that it is easy for anyone to use. Link: GitHub

ZEPPO

Roughly a year ago, I set myself the challenge of implementing a set of functions that could convert between numbers of different bases. I implemented this in Dart and decided to re-write this implementation in Javascript. The Dart implementation can be found here. The bases this library handles are: base 10, base 2, and base 16. Link: GitHub

VULCHECK

Always seeking intellectual challenges, I implemented my own algorithm to check the strength of a password in Dart, roughly a year ago. Vulcheck is my port of this package to the Javascript ecosystem. The name is a combination of the words Vulnerability and Checker. You can find the original algorithm here. Link: GitHub

MY RUST PROJECTS

Here is a selection of three libraries that I’ve developed in Rust.

ANGEL MARKUP LANGUAGE

Some months ago, I was building a static-site-generator and needed a data-storage format which allowed comments and was light-weight. While I did use JSON for some time, I didn’t like using it too much. Angel Markup Language is that format. This implementation is the compiler for this data format implemented in Rust. You can cross-compile any AML file to TOML, YAML, or JSON. Link: GitHub

CLEASY

When I was getting started with learning Rust, I used to manually parse command-line arguments. Tired of repeating myself time and again, I decided to write a library to write my parsing code once and use it everywhere. I know “clap” would have been option that seemed too daunting at the time. Link: GitHub

FLEK

Having implemented my algorithm for checking passwords in Dart and Javascript, I thought I should probably implement that same algorithm in Rust, too. Why Flek? The name is a combination of the words Flaw and Che(c)k. Link: GitHub

MY VUE.JS PROJECTS

Here is a small selection of light web applications I implemented in Vue.js 3 and SCSS.

CERAMIC

Since I was trying to figure out the algorithm for validating IMEI numbers for a uni course, I thought I’d write a small app that solves this interesting problem. This little app checks whether the IMEI number of any one of your mobile devices is valid or not by computing the check digit and comparing this to the actual check digit. The “Luhn Algorithm” is used to do this. Link: GitHub

VCHECK

Since I’d already implemented my own algorithm for checking password security in Rust, Dart, and Javascript, I thought I’d provide an easy-to-use website for anyone to check their password on the web. You type your password in and the website tells you whether your password is secure or not. Link: GitHub

CMATCH

As an artist, I often find myself wondering about good color schemes. To address this problem, I wrote a small web app that would provide me with a complementary color for a color of my choice. Link: GitHub
