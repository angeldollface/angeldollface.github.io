## OH! YOU WANT TO TRY IT OUT FOR YOURSELF? :see_no_evil: :heart_on_fire:

Make sure you have the following tools installed and available from the command line:

- Node.js
- Git

Once you have these tools installed, execute these steps:

- 1.) Get the source code:

```bash
$ git clone https://github.com/angeldollface/angeldollface.github.io
```

- 2.) Change directory into the source's root directory:

```bash
$ cd angeldollface.github.io
```

- 3.) Install the project's dependencies:

```bash
$ npm install
```

- 4.) If you want to view this site locally, run the development server:

```bash
$ npm run dev
```

- 5.) If you'd like to compile this project and inspect the compiled files, run this command (They will be in a directory called `dist`.):

```bash
$ npm run build
```
