## CERAMIC

***A small Vue.js 3 app to validate IMEI numbers.***

Since I was trying to figure out the algorithm for validating IMEI numbers for a uni course, I thought I'd write a small app that solves this interesting problem. This little app checks whether the IMEI number of any one of your devices is valid or not by computing the check digit and comparing this to the actual check digit. The "Luhn Algorithm" is used to do this. Link: [GitHub](https://github.com/angeldollface/ceramic)

## VCHECK

***Check your passwords from anywhere.***

A year ago, I wrote an algorithm to check the security of passwords, any passwords. I implemented it in Dart, the implementation of which can be found [here](https://angeldollface/securitycheck). I recently decided to rewrite the implementation in ECMA Script, a.k.a. Javascript. I did that, but then decided to make a shiny little web application out of the Javascript implementation. This is that shiny web application. The ES implementation can be found [here](https://github.com/angeldollface/vulcheck). My one true love, having been Python, has shifted to Rust and the implementation of the same algorithm can be found [here](https://angeldollface/flek). Link: [GitHub](https://github.com/angeldollface/vcheck)

## CMATCH

***A small color matching app written in Vue.js 3.***

As an artist, I often find myself wondering about good color schemes. To address this problem, I wrote a small web app that gives you a complementary color in hex from a source color in hex. Link: [GitHub](https://github.com/angeldollface/cmatch)
